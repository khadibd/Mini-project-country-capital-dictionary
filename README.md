\# Country-Capital Dictionary



\## Description

This is a simple Python program that allows users to look up the capital of a country.  

If the country is not in the dictionary, the user can add it dynamically.  



It demonstrates:

\- Python dictionaries

\- Loops and conditions

\- User input handling

\- Dynamic data update



---



\## Features

\- Look up a capital by entering a country name

\- Add new countries and capitals if they are missing

\- Display the updated dictionary at the end

\- Simple interactive console program
